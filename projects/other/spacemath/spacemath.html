SpaceMath

SpaceMath is a math learning web application developed by a team of 5 students. In this project, I was responsible for developing .NET backend with API, including SQL database integration using Entity Framework. Application includes gamification and social features like friends system and top users rankings. Development of this application taught me a lot about a .NET framework, as well as collaborative work using GitHub and Jira.
