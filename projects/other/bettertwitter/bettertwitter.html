BetterTwitter

BetterTwitter is an imitation of a social media web application focusing on security. The application features:

  • containerized backend (Docker),
  • reverse proxy configuration,
  • HTTPS encryption,
  • two-factor authentication (2FA),
  • password hashing,
  • message signing,
  • login attempt logging.
